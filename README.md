Mini projeto desenvolvido com foco na consolidação de fundamentos do React.

## 🔗 Deploy

Acesse o projeto online:

## 🚀 Sobre o Projeto

O **ZooCardsTech** é uma aplicação simples que renderiza cards de animais de forma dinâmica a partir de um array de objetos.

O objetivo foi reforçar o entendimento sobre como dados estruturados podem ser transformados em interface utilizando o paradigma declarativo do React.

---

## 🧠 Conceitos Aplicados

- ✔️ Componentização
- ✔️ Criação de componente reutilizável (`Card`)
- ✔️ Props para passagem de dados
- ✔️ Estrutura de dados com array de objetos
- ✔️ Renderização dinâmica com `.map()`
- ✔️ Uso de `key` em listas
- ✔️ Organização de layout com Flexbox
- ✔️ Responsividade com Media Queries

---
